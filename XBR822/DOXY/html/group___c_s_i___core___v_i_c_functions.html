__STATIC_INLINE uint32_t csi_vic_get_prio ( int32_t  IRQn)

Get Interrupt Priority.

Read the priority of an interrupt. The interrupt number can be positive to specify an external (device specific) interrupt, or negative to specify an internal (core) interrupt.

参数:
[in]IRQnInterrupt number.
返回:
Interrupt Priority. Value is aligned automatically to the implemented priority bits of the microcontroller.
