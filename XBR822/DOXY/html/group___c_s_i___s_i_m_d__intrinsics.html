详细描述

Access to dedicated SIMD instructions
Single Instruction Multiple Data (SIMD) extensions are provided to simplify development of application software. SIMD extensions increase the processing capability without materially increasing the power consumption. The SIMD extensions are completely transparent to the operating system (OS), allowing existing OS ports to be used.

Halfword packing instruction. Combines bits[15:0] of val1 with bits[31:16] of val2 levitated with the val3.

Combine a halfword from one register with a halfword from another register. The second argument can be left-shifted before extraction of the halfword.

参数:
[in]val1first 16-bit operands
[in]val2second 16-bit operands
[in]val3value for left-shifting val2. Value range [0..31].
返回:
the combination of halfwords.
评论:
res[15:0] = val1[15:0]
res[31:16] = val2[31:16] << val3
